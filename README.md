How to use Shortcut Kata repo? 
===================================

1. Clone the repository
2. Start a daily habit of mastering IntelliJ Shortcuts!: 
   1. Pull new changes
   2. Go to [Main.kt](./src/main/kotlin/Main.kt)
   3. Do all the exercises
   4. Reset the repo 
   5. Repeat every day 

And that's pretty much it.
